# ET Money Mentor

## Overview
ET Money Mentor is an application designed to provide insights and guidance on financial management, investments, and personal finance planning. It aims to empower users with the knowledge and tools required to make informed financial decisions.

## Features
- User-friendly interface for easy navigation
- Comprehensive financial advice tailored to individual needs
- Interactive charts and insights for investment tracking
- Robust security measures to protect user data
- Regular updates with new features and financial trends

## Architecture
The application's architecture follows the MVC (Model-View-Controller) pattern to separate concerns effectively. This ensures that the application is scalable, maintainable, and easier to manage.

## Installation
To run the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Rickyy-Sam07/ET-Money-Mentor.git
   ```

2. Navigate to the project directory:
   ```bash
   cd ET-Money-Mentor
   ```

3. Install the required dependencies:
   ```bash
   # For frontend
   cd frontend
   npm install

   # For backend
   cd ../backend
   npm install
   ```

4. Start the application:
   ```bash
   # For frontend
   cd frontend
   npm start

   # For backend
   cd ../backend
   npm start
   ```
